How to Use:

  1. Select a color from the palette above
  2. Click on any sticker to change its color except middle sticker of face
  3. Use Solve to get the solution for the current cube state
  4. Hold the cube in position as the above cube
  5. Apply the obtained solution steps to solve the cube
  6. Use Scramble to generate a random cube configuration
  7. The solution will appear below the cube (or an error if the cube is invalid)
